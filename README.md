# AI Knowledge Base

A professional AI-driven platform for intelligent document analysis, utilizing a RAG (Retrieval-Augmented Generation) architecture.

## 🚀 Features
* **Intelligent RAG Engine**: Powered by LangChain and OpenAI.
* **FastAPI Backend**: Robust, asynchronous API for document processing.
* **Next.js Frontend**: Modern, responsive UI with Tailwind CSS.
* **ChromaDB**: Efficient local vector storage for document retrieval.

## 🛠 Tech Stack
* **Backend**: Python 3.13, FastAPI, LangChain, OpenAI API, ChromaDB.
* **Frontend**: TypeScript, Next.js 15, Tailwind CSS, Axios.
